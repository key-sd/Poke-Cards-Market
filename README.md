# PokéCards Market by Keyri
Aplicación web para explorar y comprar cartas coleccionables de Pokémon con pagos simulados vía PayPal Sandbox.

----------------------------------------------------

## Requisitos
- [Node.js](https://nodejs.org) v18 o superior

----------------------------------------------------

## Cómo correrlo

Abre la terminal del proyecto y ejecuta:

    npm install
    npm start

Luego abrir en:
    `http://localhost:3000`
